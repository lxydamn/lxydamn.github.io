Spring Web 加密组件

发表于2025-08-21|更新于2025-08-21|业务增强
|浏览量:

简要介绍

本组件主要使用到了 SpringWeb 中 RequestParamMethodArgumentResolver,PathVariableMethodArgumentResolver等参数解析器和自定义Jackson序列化器，对业务对象进行加密/解密。该过程完全在后端中完成，可以实现前端无感的数据加密脱敏。

Github: https://github.com/lxydamn/zeon/tree/master/zeon-encrypt

使用方法

  1. 添加依赖
  1. 添加注解

只需要在实体类和 Controller 中添加@Encrypt注解即可实现对返回值和请求参数进行加密了！

原理简介

graph LR
A[客户端发送请求] --> B{请求类型}
B -->|@RequestParam| C[RequestParam 解析器]
B -->|@PathVariable| D[PathVariable 解析器]
B -->|@RequestBody| E[Jackson 反序列化]
C --> F[参数解密]
D --> F
E --> F
F --> G[业务逻辑处理]
G --> H[生成响应对象]
H --> I[Jackson 序列化]
I --> J[字段加密]
J --> K[返回加密响应]
